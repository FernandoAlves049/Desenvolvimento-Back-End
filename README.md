# Desenvolvimento Back-End

Repositório com atividades da disciplina de desenvolvimento back-end.

## Objetivo

Este README é apenas uma visão geral da matéria.
Cada atividade possui sua própria documentação dentro da respectiva subpasta.

## Organização

As entregas estão separadas por aula/tema dentro do repositório.

```text
Desenvolvimento-Back-End/
├── README.md
└── Aula IV/
	└── Calculadora/
		└── Calculadora/
			├── pom.xml
			├── README.md
			├── docs/
			│   ├── README.md
			│   └── ENDPOINTS.md
			└── src/
```

## Como usar este repositório

1. Acesse a pasta da aula desejada.
2. Leia o README local da atividade para instruções de execução, endpoints e detalhes técnicos.

## Observação

Se novas aulas forem adicionadas, a documentação principal continuará sendo mantida dentro de cada subpasta.
